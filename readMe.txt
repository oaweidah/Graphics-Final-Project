The following project successfully implements the Marching Cubes Algorithm which is used to generate Triangle meshes from a scalar field.
Code Language: C++
OpenGL Support: GLFW and GlEW
Matrix Operations: GLM
PLY files and Screenshots of the produced 3D surfaces are included in the zip file.
WASD keys can be used to rotate the FOV and zoom inte/out of the surface. Left mouse button can also be used to do the same.
